# Discord Music Bot (Disnake + yt-dlp)

Макет дискорд-бота для проигрывания музыки.  
Бот работает на основе скачивания песен во время воспроизведения, а затем удаляет их.  
Реализованы основные функции управления воспроизведением.

## Функции
- Запуск трека (`~play <url>`)
- Пропуск текущего трека (`~skip`)
- Пауза (`~pause`)
- Продолжить воспроизведение (`~resume`)
- Остановка (`~stop`)

## Источники музыки
- **SoundCloud** — полностью реализовано и работает
- **YouTube** — есть готовый макет (пока отключено из-за ограничений YouTube)
- **Spotify** — не поддерживается

## Особенности
- Очередь треков с автоматическим воспроизведением следующего
- Логирование ошибок в файл `logs.txt`
- Простая структура кода для дальнейшего расширения

## Установка и запуск
1. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
2. Скачайте и установите FFmpeg и добавьте его в PATH
3. (Опционально) Скачайте и установите Node.js, если планируете использовать поддержку YouTube
4. Создайте папку music в корне проекта
5. Создайте файл config.py и впишите туда:
   ```
   TOKEN = "Ваш токен"
   ```
   Токен можно взять на Discord Developer Portal
